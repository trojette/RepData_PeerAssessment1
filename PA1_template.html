e) This done, I had a closer look at the data and noticed that data was missing for 8 whole days. The data for each 5-minute interval in these 8 days were replaced by the average per according 5-minute interval. So these missing days data are identical.

However, this operation should not have had an effect on the global average.

Indeed, you can add as many times the average of a set of values to this set of values, it won't alter the average, as the average is: \[A = \frac{S}{n}\] but also \[A = \frac{(S + k*A)}{(n+k)}\]

(where \(S\) is the sum of a set of \(n\) values, \(A\) being their average and \(k\) the additional number of occurrences of \(A\) you want to add to the set)

As a conclusion, as announced by Dr. Peng, NAs have an impact on computations done using the mean and median functions.
